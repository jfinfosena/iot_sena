# ¿Qué es la Domótica? - Presentación Rápida

## 🏠 Definición Simple

**Domótica** = **Domus** (casa en latín) + **Automática**

> La domótica es la tecnología que hace que tu casa sea **inteligente** y **automática**.

## 🔧 Componentes Básicos

### Hardware
- **Microcontroladores**: ESP32, Arduino
- **Sensores**: Temperatura, movimiento, luz
- **Actuadores**: LEDs, motores, relés
- **Conectividad**: WiFi, Bluetooth

### Software
- **Protocolos**: MQTT, HTTP
- **Aplicaciones móviles**
- **Plataformas en la nube**
